## To run locally

######  Install dependencies
$ npm install --legacy-peer-deps

###### Force audit fix
$ npm audit fix --force

###### Start the development server
$ npm start

## To Deploy

###### Using vercel CLI
$ vercel build
$vercel deploy --prebuilt
